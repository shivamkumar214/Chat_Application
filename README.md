# 💬 Real-Time Chat Application

A full-stack Real-Time Chat Application built using modern web technologies like React.js, Node.js, Express.js, Socket.IO, and MongoDB.
This application allows users to communicate instantly without refreshing the page. It supports secure authentication, real-time messaging, online/offline user status, and image sharing functionality.

## ✨ Features

- 🔐 User Authentication (Login/Register)
- ⚡ Real-Time Messaging using Socket.IO
- 🟢 Online/Offline User Status
- 🖼️ Image Sharing Support
- 📱 Responsive Design (Mobile + Desktop)
- ☁️ Cloud Image Storage using Cloudinary
- 🔒 JWT Authentication
- 🌐 Full Stack Deployment

1️⃣ Clone the repository
git clone https://github.com/your-username/your-repo-name.git
2️⃣ Install dependencies
Frontend
cd client
npm install
Backend
cd server
npm install
3️⃣ Setup Environment Variables

Create .env file inside the server folder:

PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
4️⃣ Run the project
Frontend
cd client
npm run dev
Backend
cd server
npm start
